Zebus.Directory has been installed !
====================================

The executable Zebus Directory is available in your packages directory : packages/Zebus.Directory.[version]/tools/Abc.Zebus.Directory.exe.

By default your directory will start on localhost using the port 129 and environment "Demo". 
You can update these settings by editing the file packages/Zebus.Directory.[version]/tools/Abc.Zebus.Directory.exe.config.

Remember that this directory is for debugging and demo purpose, it comes without any persistent storage, its state will be lost on restart.

For me informations about the directory, please refer to https://github.com/Abc-Arbitrage/Zebus/wiki/Directory
